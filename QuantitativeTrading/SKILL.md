# A股量化选股与交易 Skill

## 触发词

当用户说"分析A股"、"选股"、"今日推荐"、"明天买什么"等关键词时，按以下流程执行。

---

## A股开市时间

| 时段 | 时间 |
|------|------|
| 早盘集合竞价 | 9:15 - 9:25 |
| 早盘连续竞价 | 9:30 - 11:30 |
| 午盘连续竞价 | 13:00 - 14:57 |
| 尾盘集合竞价 | 14:57 - 15:00 |

**最早可下单时间：9:15（集合竞价）**，但新手建议在 **9:30-10:00** 之间观察5-10分钟再买入，避免集合竞价波动。

**最佳买入时段：**
- 9:30-10:00：开盘观察期，看是否有回调机会
- 10:00-10:30：趋势基本确立，此时买入较稳妥
- 14:00-14:30：午后观察，主力资金方向明确

**最佳卖出时段：**
- 10:00-10:30：如果冲高到位，可止盈
- 14:30-14:55：尾盘确认，如果走势不及预期止损

⚠️ 周末和法定节假日不开市。可在"日历"中查下一个交易日。

## 新手买卖指南

### 推荐价格怎么用？

每个推荐会给出4个关键价位，用具体例子说明：

> 例：推荐圆通速递，买入价21.07-21.93，目标价21.81，止损价20.42

| 价位 | 含义 | 实操 |
|------|------|------|
| **买入价区间** | 建议在这个价格范围内买入 | 明天开盘后看实时股价，落在这个区间就下单买入 |
| **目标价** | 预期涨到的价位，到达后卖出 | 可提前挂限价卖出单（条件单），到价自动卖出 |
| **止损价** | 跌破必须卖出，保住本金 | 可提前挂止损单，跌破自动卖出 |

### 具体操作步骤

**Step 1：当晚看推荐**
- 看完推荐报告，记住明天要买的股票代码和价格区间

**Step 2：第二天9:15-9:25（集合竞价）**
- 打开券商APP，搜索股票代码
- 观察集合竞价价格，是否在建议买入价区间内
- ⚠️ 如果开盘价远高于买入区间（比如涨停开盘），**不要追**，等回调或放弃

**Step 3：9:30后下单买入**
- 如果开盘价在买入区间内 → 直接市价买入或挂限价单
- 如果开盘价低于买入区间 → **更好**，可以更低价格买入
- 如果开盘价高于买入区间3%以上 → **放弃**，追高容易亏

**Step 4：立即设置止损单**
- 买入后**立刻**在APP挂条件卖出单：股价跌破止损价 → 自动卖出
- 这是铁律，不要心存侥幸

**Step 5：设置目标价卖出单**
- 同时挂条件卖出单：股价到达目标价 → 自动卖出（或减半卖出保利润）
- 更激进做法：到目标价先卖一半，剩余部分看能否继续涨

**Step 6：持有期间**
- 每天收盘后看一次，关注是否出现卖出信号（放量滞涨、MACD死叉等）
- 持有天数参考推荐中的"持有周期"

### 仓位怎么控制？

假设你有10万资金：

| 仓位 | 金额 | 单只股票买入量 |
|------|------|----------------|
| 1/4仓 | 2.5万 | 保守试探，首次建仓 |
| 1/3仓 | 3.3万 | 看好时加仓到这个仓位 |
| 1/2仓 | 5万 | 非常看好（一般不超过此仓位） |

**核心原则：永远不要全仓一只股票！**

A股买入必须是100股的整数倍（1手=100股）。根据你的资金量：

| 总资金 | 1/4仓 | 建议同时持有 | 说明 |
|--------|-------|-------------|------|
| 3,000元 | 750元 | 1只 | 最小起步，买100股低价股 |
| 5,000元 | 1,250元 | 1-2只 | 各买100股，风险可控 |
| 10,000元 | 2,500元 | 2-3只 | 每只100-200股 |
| 50,000元 | 12,500元 | 3只 | 每只1/3仓 |

**新手建议从5000元以内起步**，每只只买100股（1手），先熟悉流程再逐步加仓。

### 卖出时机

| 情况 | 操作 |
|------|------|
| 股价到达目标价 | 卖出（至少卖一半锁利润） |
| 股价跌破止损价 | **立刻卖出**，不犹豫 |
| 持有天数已到上限 | 卖出，不管盈亏 |
| 出现放量滞涨 | 减仓或清仓 |
| 出现利好但股价不涨 | 小心，可能利好出尽，减仓 |

## 完整工作流

### Phase 1: 数据获取

运行 `python3 data_fetcher.py`，获取：
- 当日全A股行情（涨跌幅排行、成交额、换手率）
- 三大指数K线数据（上证、深证、创业板）
- 涨幅前30股票K线数据（近60个交易日）

### Phase 2: 新闻热点获取

运行 `python3 news_fetcher.py`，获取：

**国内新闻：**
- 东方财富财经要闻
- 新浪财经新闻

**国际新闻：**
- CNBC（美股/全球经济）
- Google News（中国股市相关）
- MarketWatch（全球市场）

- 提取热点关键词与权重（含中英文关键词）

### Phase 3: 技术面分析

运行 `python3 analyzer.py`，对候选股票进行技术分析：
- **趋势判断**: MA5/MA10/MA20/MA60 多头排列
- **量价配合**: 放量上涨、缩量回调、量价背离
- **MACD**: 金叉/死叉、红绿柱变化、背离
- **KDJ**: 超买超卖区域、金叉信号
- **RSI**: 超买(>70)/超卖(<30)区域
- **布林带**: 突破上轨/下轨、缩口变盘信号
- **支撑压力位**: 关键价位识别

### Phase 4: 综合研判

读取 Phase 1-3 的输出数据（JSON/CSV），结合技术面评分、新闻面热点和**国际形势**，深度分析后给出最终推荐。

**筛选逻辑：**
1. 技术面评分 >= 3 的股票优先考虑
2. 新闻热点与个股所属板块形成共振的加分
3. 排除涨停/接近涨停的股票（追高风险大）
4. 排除RSI超买(>75)且放量的股票（短线过热）
5. 国际重大利空（战争、制裁、美联储加息）时降低仓位
6. 优先选择：有题材驱动 + 技术面刚启动 + 量能配合 = 最佳买点

### Phase 6: 交易记录

**两个CSV，各司其职：**

1. **`recommendations.csv`** — 我的推荐记录（每次推荐自动写入）
   - 日期、股票代码、名称、建议买入价区间、目标价、止损价、仓位、持有天数、推荐理由、评分

2. **`trades.csv`** — 你的实际交易记录（你反馈后我更新）
   - 日期、股票代码、名称、买入价、买入时间、卖出价、卖出时间、数量、成本、盈亏、盈亏比例、是否触及止损、备注

### Phase 7: 迭代优化

- 每周五进行周度复盘，统计：胜率、平均盈亏比、最大回撤
- 在推荐报告中记录策略有效性（哪些信号有效、哪些失效）
- 根据复盘结果调整参数（均线周期、止盈止损比例、评分权重）

## 约束与风控

1. **单只股票仓位不超过总资金1/3**
2. **总持仓不超过总资金的2/3**
3. **严格止损，亏损超过5%无条件卖出**
4. **追涨不追高**：只在调整到位时介入，不追连续涨停
5. **避开ST股、停牌股、上市不足60天的新股**
6. **优先选择有题材驱动+技术面共振的股票**
7. 推荐不超过3只股票，宁缺毋滥
8. 如无符合条件的股票，明确告知"今日无推荐，空仓观望"
9. **国际重大利空**（战争升级、制裁、美股大跌等）时降低仓位至1/4以下
